Realtime Weather UI

Thiết kế lại cách bạn xem thời tiết mỗi ngày.

Tìm kiếm thành phố với OpenWeather Geocoding API, chạm vào kết quả để xem nhiệt độ, cảm giác, gió và độ ẩm theo thời gian thực. Toàn bộ được dựng bằng TailwindCSS và Axios.

Demo giao diện tìm kiếm

Có thể nhập dạng: city hoặc city,countryCode.

Sử dụng Axios gọi trực tiếp OpenWeather Geocoding API /geo/1.0/direct.

UI hiện đại

Thiết kế glassmorphism, gradient mềm, tối ưu cho cả desktop lẫn mobile.

100% Tailwind

Toàn bộ layout & style sử dụng utility class của TailwindCSS (CDN).

Sẵn sàng API

Dễ dàng gắn Axios & Open-Meteo API ở bước sau mà không cần chỉnh lại UI.

Stack sử dụng

HTML TailwindCSS (CDN) Axios (cho bước tiếp theo)

Kết quả geocoding (OpenWeather)

Nhập tên thành phố và bấm Tìm vị trí để xem danh sách kết quả trả về từ API.

Giao diện đã được kết nối trực tiếp tới OpenWeather API thông qua Axios. Bạn có thể tùy biến thêm hoặc tách component theo ý thích.
