WeatherApp

Theo dõi thời tiết thời gian thực cho mọi thành phố.

Một landing page tối giản nhưng hiện đại cho ứng dụng xem thời tiết. Được xây dựng hoàn toàn bằng HTML và TailwindCSS, sẵn sàng để bạn kết nối Axios và API thời tiết sau này.

Demo giao diện tìm kiếm

Ví dụ: Hanoi, Ho Chi Minh City

Phần chức năng (gọi API bằng Axios, hiển thị thời tiết) sẽ được bổ sung ở bước tiếp theo.

UI hiện đại

Thiết kế glassmorphism, gradient mềm, tối ưu cho cả desktop lẫn mobile.

100% Tailwind

Toàn bộ layout & style sử dụng utility class của TailwindCSS (CDN).

Sẵn sàng API

Dễ dàng gắn Axios & Open-Meteo API ở bước sau mà không cần chỉnh lại UI.
